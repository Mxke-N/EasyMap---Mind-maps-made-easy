Mind Map Controls

  • Right-click: Create bubble
  • Left-click drag: Move bubble
  • Scroll while dragging: Resize bubble

  • Right-click hold: Connect bubbles
  • Left-click while connecting: Toggle line type
  • Middle-click: Delete bubble/line
  • Scroll wheel: Zoom in/out
